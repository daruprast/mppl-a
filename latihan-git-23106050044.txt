sebutkan jenis versio control dalam sdlc
1. local version control system
menyimpan file secara local di masing masing komputer developer, setiap perubahan disimpan di directori local
2. centralized version control system
semua file dan riwayat perubahan disimpan di server pusat, developer harus terhubung ke server untuk mengambil atau mengirim perubahan.
3. distributed version control system
setiap developer memiliki salinan lengkap repositori,perubahan bisa dilakukan lokal, lalu disinkronkan ke server bersama. contoh git
