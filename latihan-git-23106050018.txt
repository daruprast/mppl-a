4. Sebutkan dan Jelaskan jenis version control yang digunakan dalam pengembangan perangkat lunak
jawaban: 
- Local Version Control: Menyimpan revisi file di komputer lokal, contohnya RCS, cocok untuk proyek individu tetapi rawan kehilangan data.
- Centralized Version Control (CVCS): Menggunakan satu server pusat untuk menyimpan semua versi, contohnya Subversion (SVN), memungkinkan kolaborasi tetapi tergantung pada koneksi ke server.
- Distributed Version Control (DVCS): Setiap pengembang memiliki salinan penuh repositori, contohnya Git dan Mercurial, lebih fleksibel, mendukung kerja offline, dan aman terhadap kerusakan server pusat.
