4.  - GIT
        yaitu version control yang setiap pengguna memiliki salinan repository di local, sehingga tidak tergantung sepenuhnya pada server.
    - Subversion
        yaitu version control yang menyimpan semua versi file di server sehingga pengguna harus terkoneksi ke server untuk melakukan operasi
    - Mercurial
        mirip dengan GIT namun tampilan lebih sederhana
    - CVS
        merupakan version control yang paling tua yang digunakan untuk melacak perubahan file
